Use Case: Filter By Seller
Context: Discount set has already been found/loaded
Main Scenario:
1. User selects “Seller” from filter-by dropdown
2. Input textbox’s label changes to prompt user to enter the seller’s name
3. User enters name of store
4. Discounts are found that fit the filtered parameters
5. The found discounts are displayed to the user
Alternative Flow: 
4a. No discounts from store are found
4a1. User is displayed a message conveying that no matching deals have been found
